# Blog Semântico

Um layout de blog limpo, moderno e semântico, desenvolvido com foco em estrutura HTML sólida e estilização via CSS.

## 🚀 Tecnologias Utilizadas

*   **HTML5**: Estrutura semântica do conteúdo.
*   **CSS3**: Estilização visual e layout.
    *   **Flexbox**: Utilizado extensivamente para o sistema de grid (navbar, container principal, sidebar).
    *   **Responsividade**: Adaptação para dispositivos móveis (`@media query` para telas < 450px).

## 🎨 Design e Estilo

O projeto segue uma identidade visual minimalista e legível:

*   **Tipografia**: Utiliza a fonte *"Georgia"* para um visual clássico de blog/jornal.
*   **Paleta de Cores**:
    *   Texto principal: Cinza escuro (`#444`).
    *   Destaques/Links: Azul vibrante (`#37a1f7`).
    *   Bordas e fundos secundários: Cinza claro (`#ddd`).

## 📂 Estrutura do Layout

O layout é dividido nas seguintes seções principais:

1.  **Navbar**: Cabeçalho com logotipo e links de navegação alinhados via Flexbox.
2.  **Container Principal**: Área central que agrupa os posts e a barra lateral.
    *   **Posts Area**: Lista de artigos contendo imagem de destaque, título, descrição e autor.
    *   **Sidebar**: Barra lateral funcional contendo:
        *   Barra de busca (Search bar).
        *   Lista de categorias.
        *   Nuvem de tags (Tags cloud).
3.  **Footer**: Rodapé centralizado para copyrights e informações finais.

## 🔧 Como Executar

1.  Faça o clone deste repositório.
2.  Abra o arquivo `index.html` em seu navegador de preferência.
